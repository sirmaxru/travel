| | |
|---|---|
| **Название** | 锅德火锅 (Guode Hot Pot) |
| **Адрес** | 自忠路338号, Xintiandi |
| **Тип** | Хот-пот (火锅) |
| **От отеля** | ~500м пешком (5 мин) |
| **Цены** | Выше среднего (~¥150-200/чел) |

### Как дойти от отеля
```
Выйти из отеля (淡水路214弄)
→ Повернуть направо на 淡水路
→ Дойти до 自忠路 (Zizhong Road)
→ Повернуть налево
→ Ресторан слева по улице
```

### Что заказать
- 🍲 **Бульон**: рекомендую 鸳鸯锅 (юаньян) — половина острый, половина нет
- 🥩 Нарезка говядины/баранины
- 🥬 Овощи, грибы, тофу
- 🍜 Лапша в конце

👧 **Для детей**: не острый бульон + мясные шарики + овощи

## 🚶 NANJING ROAD (南京路)

Между Starbucks и M&M's — знаменитая **Nanjing Road Pedestrian Street** (南京路步行街):

- 🛍️ Главная торговая улица Шанхая
- 📏 ~5 км от Nanjing West до Бунда
- 🏬 Магазины, торговые центры, рестораны
- 🚶 Можно пройти пешком (~30-40 мин) или подъехать на метро

**Маршрут пешком:**
```
Starbucks (南京西路789号)
   ↓ ~15 минут пешком
People's Square (人民广场)
   ↓ ~20 минут пешком
M&M's World (南京东路829号)
   ↓ ~10 минут пешком
The Bund (外滩)
```

💡 Если устали — Line 2 соединяет все точки!

## 🌅 НАБЕРЕЖНАЯ БУНД (外滩)

| | |
|---|---|
| **Название** | 外滩 (Wàitān / The Bund) |
| **Где** | Восточный конец Nanjing Road, набережная реки Хуанпу |
| **Закат** | ~17:10 (4 января) |
| **Огни Пудуна** | С 18:00 до 22:00 |

### Что увидите
- 🏛️ Историческая архитектура европейских банков (1920-30-е годы)
- 🌃 Панорама Пудуна через реку:
  - Oriental Pearl Tower (东方明珠)
  - Shanghai Tower (上海中心大厦) — 632м, 2-е в мире
  - Jin Mao Tower (金茂大厦)
  - Shanghai World Financial Center (上海环球金融中心)
- 🚢 Прогулочные корабли по реке

### Лучшие фото-точки
- 📸 У быка на Бунде (Bund Bull)
- 📸 Смотровая площадка у Chen Yi Square
- 📸 Вид на Oriental Pearl с набережной

⚠️ **Важно**: В январе холодно на набережной! Оденьтесь тепло, ветер с реки.
